“寻城记”分栏的内容

发表于 2020-05-14 | 分类于 寻城记 | |

　　2019年年底，我去公司总部培训，花了周末两天时间逛了南京的一些著名景点。从此爱上了做攻略、写计划并实施的感觉，并打算开一个“寻城记”的分栏，以游记分享为主。

　　寻城记，就是寻找城市的记忆，所以基本都是在城市范围内短途旅行，时间跨度不会超过两天，主要供城市上班族周末游览参考。游记的内容会包括游览顺序、主要交通工具（地铁、公交为主）、地点介绍。

　　因为是游记，所以不可避免的会涉及到图片，我目前还没有想好怎么去写具体的文章。一方面，我并不希望在自己的博客内出现大量的图片，影响文字的阅读；另一方面，作为一个经常要做PPT的人，我深知“一图二表三文字”的原理，图片对人的刺激是最大的，也是最能吸引人眼球的。

　　我会考虑好游记的呈现方式的，敬请期待。
